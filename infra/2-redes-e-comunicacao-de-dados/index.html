    2. Redes e Comunicação de Dados#

    1. Introdução e Contextualização#

    A infraestrutura de Tecnologia da Informação (TI) é um pilar fundamental para o funcionamento das organizações modernas, e as redes e comunicação de dados formam a espinha dorsal dessa infraestrutura. Este tópico abrange desde os princípios básicos de comunicação de dados até as tecnologias mais avançadas utilizadas para conectar dispositivos e sistemas de forma segura e eficiente. A compreensão desses conceitos é crucial para profissionais que almejam cargos de alto nível, como Consultor Legislativo e Auditor Fiscal, onde a tecnologia desempenha um papel estratégico.

    2. Desenvolvimento Teórico Aprofundado#

    Protocolos de Comunicação#

    • TCP (Transmission Control Protocol): Protocolo orientado à conexão, garantindo a entrega de pacotes na ordem correta e sem erros.
    • UDP (User Datagram Protocol): Protocolo sem conexão, utilizado para transmissões que não requerem confirmação de entrega.
    • SCTP (Stream Control Transmission Protocol): Combina características do TCP e UDP, oferecendo múltiplos fluxos de dados dentro de uma única conexão.

    Segurança#

    • TLS (Transport Layer Security) e SSL (Secure Sockets Layer): Protocolos criptográficos que fornecem comunicações seguras na Internet para serviços como email, navegação web e outros.

    Redes#

    • OSPF (Open Shortest Path First) e BGP (Border Gateway Protocol): Protocolos de roteamento utilizados em redes IP.
    • DNS (Domain Name System): Sistema que traduz nomes de domínios para IP.
    • DHCP (Dynamic Host Configuration Protocol): Protocolo que atribui automaticamente IPs e outras configurações de rede a dispositivos.
    • ICMP (Internet Control Message Protocol): Utilizado para enviar mensagens de erro e operacionais.

    Transferência de Arquivos e Comunicação#

    • FTP (File Transfer Protocol), SFTP (SSH File Transfer Protocol), SSH (Secure Shell): Protocolos para transferência segura de arquivos e acesso remoto.
    • HTTP (Hypertext Transfer Protocol) e HTTPS (HTTP Secure): Protocolos para transmissão de documentos na web.
    • SMTP (Simple Mail Transfer Protocol), IMAP (Internet Message Access Protocol), POP3 (Post Office Protocol 3): Protocolos para envio e recebimento de emails.

    Redes Avançadas#

    • VLANs (Virtual Local Area Networks), STP (Spanning Tree Protocol), QoS (Quality of Service): Tecnologias para segmentação de redes, prevenção de loops e garantia de qualidade de serviço.
    • SDN (Software-Defined Networking): Abordagem para redes que centraliza o controle da rede, tornando-a programável.
    • Wireless: Wi-Fi 6, WPA3 (segurança), roaming e mesh são tecnologias para redes sem fio.

    3. Visão das Bancas (Análise Estratégica)#

    As bancas como Cebraspe, FGV e FCC têm abordagens distintas na cobrança desse tema. Enquanto a Cebraspe valoriza a compreensão conceitual e a capacidade de análise crítica, a FGV pode focar em detalhes técnicos e aplicações práticas. A FCC, por sua vez, tende a equilibrar entre conceitos e práticas, com questões que testam o entendimento geral e a aplicabilidade dos conhecimentos.

    5. “O Pulo do Gato”: Atenção com Pegadinhas#

    1. Certo/Errado: “O protocolo UDP garante a entrega de pacotes na ordem correta.” - Errado. O UDP não garante ordem nem a entrega de pacotes.
    2. Certo/Errado: “SSL é uma evolução do TLS.” - Errado. TLS é uma evolução do SSL.
    3. Certo/Errado: “BGP é utilizado para roteamento interno em uma organização.” - Errado. BGP é geralmente utilizado para roteamento entre organizações (roteamento externo).

    6. Referências Sugeridas#

    • RFCs (Request for Comments): Documentos técnicos que descrevem os padrões de protocolos de Internet.
    • ISO/IEC 27001: Norma para o sistema de gestão de segurança da informação (SGSI).
    • Manual de Redação da Presidência da República: Para elaboração de documentos oficiais.
    • Documentação oficial do ITIL e COBIT: Para gestão e governança de TI.

    Este material fornece uma visão abrangente e aprofundada sobre redes e comunicação de dados, essencial para candidatos a cargos de alto nível em concursos públicos.
