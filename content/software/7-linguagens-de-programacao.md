# 7. Linguagens de programação

## 1. Introdução e Contextualização

A programação é uma habilidade fundamental no desenvolvimento tecnológico e na inovação de sistemas, softwares e aplicações diversas. Dentro deste vasto campo, as linguagens de programação são as ferramentas que permitem aos desenvolvedores criar soluções complexas e eficientes. Java, em particular, é uma linguagem de programação orientada a objetos, desenvolvida pela Sun Microsystems no início dos anos 90 e atualmente mantida pela Oracle. Sua importância no cenário atual de concursos públicos, especialmente para cargos de nível superior como Consultor Legislativo e Auditor Fiscal, onde a tecnologia da informação desempenha um papel crucial, é indiscutível.

## 2. Desenvolvimento Teórico Aprofundado

### Fundamentos e Arquitetura de Java

Java é uma linguagem de programação que se destaca pela sua portabilidade, segurança e robustez. Segue o paradigma de orientação a objetos, permitindo a criação de programas modulares e reutilizáveis. A arquitetura de Java gira em torno da Máquina Virtual Java (JVM), que permite a execução de código Java em qualquer dispositivo capaz de rodar a JVM, seguindo o princípio de "write once, run anywhere".

**Principais Características:**

- **Sintaxe semelhante a C/C++:** Facilita o aprendizado por parte de quem já conhece essas linguagens.
- **Gerenciamento automático de memória:** Através do Garbage Collector, Java automatiza o processo de alocação e desalocação de memória, reduzindo erros e vazamentos de memória.
- **Multithreading integrado:** Java fornece uma rica API para o desenvolvimento de aplicações multithreaded, permitindo um uso eficiente dos recursos do processador.

### Detalhes Específicos: Java

- **Plataforma Java:** Dividida em Java Standard Edition (Java SE), Java Enterprise Edition (Java EE) e Java Micro Edition (Java ME), cada uma destinada a diferentes tipos de aplicações, desde desktops a sistemas empresariais e dispositivos móveis.
- **APIs e Bibliotecas:** Java possui uma vasta coleção de bibliotecas padrão, cobrindo desde operações de entrada/saída até desenvolvimento web e manipulação de bancos de dados.

## 3. Visão das Bancas (Análise Estratégica)

As bancas de concursos públicos, como Cebraspe, FGV e FCC, têm abordagens distintas ao cobrar conhecimentos em Java. Enquanto a Cebraspe valoriza a compreensão profunda dos conceitos e fundamentos da linguagem, com questões que exigem análise e interpretação de blocos de código, a FGV e a FCC tendem a focar mais em aspectos práticos e específicos da API de Java e em questões de múltipla escolha.

**Diferenças Importantes:**

- **Cebraspe:** Prefere questões dissertativas ou de certo/errado que testam a compreensão conceitual e a capacidade de aplicar o conhecimento em cenários complexos.
- **FGV e FCC:** Inclinam-se para questões objetivas, muitas vezes focando em particularidades da linguagem e suas bibliotecas.

## 5. "O Pulo do Gato": Atenção com Pegadinhas

1. **Certo/Errado:** "Java é uma linguagem interpretada." (Errado, Java é compilada para bytecode, que é interpretado pela JVM.)
2. **Certo/Errado:** "Java não suporta herança múltipla de classes." (Certo, Java não suporta herança múltipla diretamente nas classes, mas permite herança múltipla de interfaces.)
3. **Certo/Errado:** "O Garbage Collector em Java elimina completamente a possibilidade de vazamento de memória." (Errado, embora reduza significativamente o risco, cenários específicos podem ainda levar a vazamentos de memória.)

## 6. Referências Sugeridas

Para um aprofundamento nos temas abordados, recomenda-se a consulta aos seguintes materiais:

- **The Java® Language Specification:** Documento oficial que especifica a linguagem Java.
- **Java SE Documentation:** Documentação oficial da Oracle para Java Standard Edition, contendo tutoriais, guias de usuário e referências da API.
- **Effective Java (Joshua Bloch):** Livro que oferece práticas recomendadas para a programação em Java, abordando desde fundamentos até aspectos avançados da linguagem.

Estes recursos fornecem uma base sólida não apenas para candidatos a concursos públicos, mas também para qualquer desenvolvedor que deseje aprofundar seus conhecimentos em Java.
