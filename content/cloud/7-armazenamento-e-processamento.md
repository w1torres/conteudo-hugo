# 7. Armazenamento e Processamento

## 1. Introdução e Contextualização

No cenário atual de concursos públicos, especialmente para cargos de alto nível como Consultor Legislativo e Auditor Fiscal, a Computação em Nuvem emerge como uma disciplina crucial. O armazenamento e processamento em nuvem, abrangendo objetos, blocos, arquivos, Data Lakes, Big Data e AI, representam a espinha dorsal da infraestrutura moderna de TI, permitindo escalabilidade, eficiência e inovação.

## 2. Desenvolvimento Teórico Aprofundado

### Fundamentos e Arquitetura

**Armazenamento:**

- **Objetos:** Armazenam dados como um objeto único, identificado por um ID único. Ideal para dados não estruturados.
- **Blocos:** Divide os dados em blocos de tamanho fixo, cada um com um endereço único. Usado em sistemas de armazenamento de baixo nível.
- **Arquivos:** Organiza dados em uma hierarquia de diretórios e arquivos, facilitando o acesso e a gestão.

**Processamento:**

- **Data Lakes:** Repositórios que armazenam grandes volumes de dados brutos em seu formato nativo até que sejam necessários.
- **Big Data:** Tecnologias e procedimentos usados para processar volumes de dados muito grandes que os sistemas tradicionais não conseguem lidar.
- **AI (Inteligência Artificial):** Simula a inteligência humana, processando grandes conjuntos de dados para aprender e tomar decisões.

### Diagrama de Arquitetura

```
[Usuário] -> [Aplicação] -> [Processamento: AI/Big Data] -> [Armazenamento: Data Lake/Objetos/Blocos/Arquivos]
```

## 5. "O Pulo do Gato": Atenção com Pegadinhas

1. **Certo/Errado:** "Data Lakes e Big Data são conceitos intercambiáveis." - **Errado.** Data Lakes referem-se ao armazenamento de dados brutos, enquanto Big Data envolve o processamento desses dados.
2. **Certo/Errado:** "O armazenamento em blocos é mais eficiente para dados não estruturados do que o armazenamento em objetos." - **Errado.** O armazenamento em objetos é projetado especificamente para dados não estruturados.
3. **Certo/Errado:** "A inteligência artificial não pode ser aplicada em Data Lakes." - **Errado.** AI pode ser usada para analisar e extrair valor dos dados armazenados em Data Lakes.
