    5. Governança e Qualidade de Dados#

    1. Introdução e Contextualização#

    A Governança e Qualidade de Dados são fundamentais para organizações que buscam a excelência na gestão de suas informações. No cenário de concursos públicos, especialmente para cargos de alto nível como Consultor Legislativo e Auditor Fiscal, o tema é abordado com profundidade, exigindo do candidato não apenas o conhecimento teórico, mas também a capacidade de aplicação prática desses conceitos.

    2. Desenvolvimento Teórico Aprofundado#

    Linhagem e Catalogação de Dados#

    A linhagem de dados refere-se ao rastreamento da origem, movimento, características e uso dos dados dentro de uma organização. A catalogação de dados, por sua vez, envolve a criação de um inventário acessível de dados disponíveis, incluindo metadados detalhados para facilitar a localização e compreensão dos dados.

    Validação e Deduplicação#

    A validação de dados é o processo de garantir que os dados inseridos em um sistema estejam corretos, completos e conforme os critérios especificados. A deduplicação é a técnica utilizada para eliminar cópias redundantes de dados repetidos, melhorando a qualidade e a eficiência do armazenamento de dados.

    Metadados, Glossários e Políticas de Acesso#

    Metadados são dados sobre dados, fornecendo informações como tipo de dado, autor, data de criação e localização. Glossários de dados são ferramentas que definem termos e conceitos de dados usados na organização. Políticas de acesso definem quem pode acessar quais dados, sob quais condições e com que finalidades.

    3. Visão das Bancas (Análise Estratégica)#

    • Cebraspe: Tende a cobrar questões que exigem compreensão conceitual e aplicação prática, com enunciados que simulam situações reais de gestão de dados.
    • FGV: Foca em questões teóricas, mas também exige a compreensão das implicações práticas dos conceitos.
    • FCC: Varia entre questões conceituais e aplicadas, com ênfase em casos práticos e interpretação de textos normativos.

    Diferença entre teoria e prática: Enquanto a teoria aborda os conceitos de forma abstrata, as bancas, especialmente o Cebraspe, buscam avaliar a capacidade do candidato de aplicar esses conceitos em cenários práticos.

    5. “O Pulo do Gato”: Atenção com Pegadinhas#

    1. Certo/Errado: “A deduplicação de dados elimina a necessidade de validação de dados.” (Errado) - A deduplicação remove dados redundantes, mas não garante a correção ou completude dos dados restantes.
    2. Certo/Errado: “Metadados são irrelevantes para a governança de dados.” (Errado) - Metadados são cruciais para entender, gerenciar e usar dados de forma eficaz.
    3. Certo/Errado: “Políticas de acesso são definidas apenas no início do ciclo de vida dos dados.” (Errado) - Políticas de acesso devem ser revisadas e atualizadas regularmente para refletir mudanças nas necessidades de negócios e na legislação.

    6. Referências Sugeridas#

    • ISO/IEC 27001: Norma internacional para a gestão da segurança da informação.
    • COBIT: Framework para a governança e gestão da TI, incluindo práticas para a governança de dados.
    • ITIL: Conjunto de práticas para a gestão de serviços de TI, com relevância para a gestão de dados.

    Estudar essas referências proporcionará uma base sólida para entender a governança e a qualidade de dados, além de preparar o candidato para as nuances e exigências das principais bancas de concursos públicos.
